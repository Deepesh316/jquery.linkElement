#linkElement jQuery Plugin #

**linkElement** is a simple jQuery plugin that allows you to attach a link to any element

##Version##
0.2.0

##Usage##
    $('span').linkElement({
		href: 'http://google.com',
		text: 'click me'
	});
    
    
##Vendor##
jQuery - [http://jquery.com](http://jquery.com)
